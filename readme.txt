Linkubaitor displays an HTML codebox for your Wordpress posts which contains the post URL and Title for anchor text. This improves search engine ranking when others grab your code and paste it to their webpages.

== Description ==

Search engines determine a website or blogs popularity according to the amount and quality of backlinks pointing to a website. The Wordpress Linkubaitor plugin displays HTML code for website visitors to copy and paste to their site. The result is more keyword based links pointing back to your site for the topics you talk about.

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload the `linkubaitor` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

Q. What is Link Bait?
A. A webpage you create designed to get visitors to link back to your website by providing high quality content and an incentive to share.
